# Password Generator 🔐

### 👨‍💻 Developer Details
* **Name:** Ishant Khare
* **Company:** CodSoft

### 🎯 Objective
To create a Python utility that generates highly secure and randomized passwords of a user-specified length.

### 🛠️ Tools Used
* Python
* `random` and `string` libraries

### 📝 Steps Performed
1. Collected user input to define the exact length of the desired password.
2. Merged character sets (uppercase, lowercase, digits, symbols) using Python's string library.
3. Randomly selected characters from the merged set to construct the final password string.

### 💡 Outcome
A customizable security tool capable of producing complex passwords to enhance user data protection.
